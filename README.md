# Mom’s Connect 

Mom’s Connect is a front-end community platform designed for mothers to connect, share experiences, and support each other in a safe and friendly environment.

## Features
- User signup with basic details
- Post text and images 
- Delete your own posts
- Posts remain after page refresh 
- Responsive and modern UI
- English & Amharic language support
- Community & Contact sections with smooth navigation
